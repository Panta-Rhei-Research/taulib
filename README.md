# 🧮 taulib

This repository is the **active contributor-facing formal code workspace** for the **τ-kernel** and its implementation.

If `formalization` is the official release-facing surface, `taulib` is the living engineering workspace.

## What lives here

- active Lean / TauLib source code
- issue-driven implementation work
- pull requests
- design discussion around formalization
- contributor-facing code evolution

## Why this repo exists

The framework needs a place where formal implementation can evolve openly without destabilizing the official public release-facing formalization surface.
