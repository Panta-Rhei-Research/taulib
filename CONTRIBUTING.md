# Contributing to TauLib

Thank you for contributing to the formal implementation layer of the Panta Rhei framework.

## What belongs here

Use this repository for:

- Lean / TauLib source code
- proof-oriented implementation work
- formal kernel extensions
- implementation-specific issues
- pull requests that improve the formal layer

## Before opening a PR

Please:

1. open or link a relevant issue/discussion where appropriate
2. keep the scope focused
3. explain the mathematical or formal motivation clearly
4. note any relation to books, parts, topics, or results if relevant

## Pull request guidance

A good PR should include:

- a clear statement of intent
- a concise technical summary
- any relevant links to discussions/issues
- tests or validation notes where relevant

## Discussion routing

- broad project questions → `community`
- exploratory research material → `research`
- release-facing formalization presentation → `formalization`

## Conduct

Please follow the organization-wide code of conduct.
